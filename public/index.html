Ik weet nog goed wanneer ik mijn eerste game had gemaakt in GameMaker Studio toen ik ongeveer 11 jaar oud was. Nu een aantal jaar later ben ik bijna afgestudeerd als Software Engineer bij de Hogeschool Utrecht en het schrijven van software is nog steeds mijn grootste passie. Het is iets waar ik met plezier elke dag uren mee bezig kan zijn.

services

Android app development

Native apps leveren de beste prestaties en ondersteunen de laatste nieuwe functionaliteiten van Android waar alleen Native apps gebruik van kunnen maken, totdat andere platforms zoals Ionic voor deze functionaliteiten worden geüpdatet.

Ionic app development

Ionic maakt het mogelijk om apps te ontwikkelen voor IOS, Android én Windows Phone tegelijkertijd. Hierdoor is de ontwikkeltijd voor een Ionic app aanzienlijk lager. Ionic heeft echter een paar kleine nadelen; de prestaties zijn namelijk minder goed en Ionic kan niet gebuik maken van de allernieuwste native functionaliteiten.

Webapplicaties

Webapplicaties zijn op veel verschillende manieren te ontwikkelen, de beste manier om een website te maken hangt af van de functionaliteiten die de website moet hebben. Zo kan AngularJS worden gebruikt voor webapplicaties met complexe functionaliteiten, of WordPress kan worden gebruik voor de minder complexe websites.

Desktop applicaties

Dit zijn applicaties die van tevoren geïnstalleerd moeten worden op een computer. Het heeft een aantal voordelen tegenover webapplicaties: alle informatie kan offline blijven, desktop applicaties leveren betere prestaties dan webapplicaties en er kan worden gebruik gemaakt van alle hardware zoals camera's en printers.

3D/2D Games

Met behulp van Unity of Phaser kunnen leuke (Applied) games ontwikkeld worden. Unity is een game-engine dat veel wordt gebruikt door zelfstandige game ontwikkelaars om 3D spellen te ontwikkelen. Phaser kan worden gebruik om (2D) HTML5 games te maken die op een website kunnen worden gezet.

Virtual reality applicaties

Ook VR kan op verschillende manieren worden ontwikkeld. Om een groot publiek te bereiken kan een VR applicatie worden ontwikkeld dat werkt op mobiele telefoons. De beste (de minst misselijk makende) ervaring krijg je echter wanneer VR wordt ontwikkeld voor de PC met een Oculus Rift, HTC Vive o.i.d.
